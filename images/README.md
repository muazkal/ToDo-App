# SimpleTodos App

A clean and minimal ToDo application that helps users manage daily tasks with ease.  
Built using HTML, CSS, and JavaScript with a focus on simplicity and usability.


## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Edit tasks
- Search tasks
- Filter completed tasks
- Dark mode toggle
- Data saved using localStorage


## UI Overview

- Simple and modern dark-themed interface
- Task list with action icons (edit and delete)
- Search bar for quick filtering
- Toggle to show or hide completed tasks


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
